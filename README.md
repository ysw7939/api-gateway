## API server

This is a simple API server

### How to run

1. Install the dependencies

```bash
$ npm install
```

2. Run the server

```bash
$ npm start
```

3. The server will be running on port 3000

---

## Docker

```bash
$ cd docker
```

```bash
$ docker-compose -f docker-compose-mysql.yml up
```
